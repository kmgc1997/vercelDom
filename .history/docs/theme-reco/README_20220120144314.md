JavaScript数据类型
---
JavaScript数据类型可分为两大类基础数据类型和引用数据类型

基础数据类型：String(字符串)、Number(数字)、Undefined(未定义)、Null(空值)、Boolean(布尔)、Symbol(独一无二)、Bigint(大数字类型)

引用数据类型：Object()
``` html
<div>
  <span>你好1</span>
</div>
```

``` js
export default {
  name: 'MyComponent',
  // ...
}
```
